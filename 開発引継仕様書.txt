プロジェクト仕様書：音声対話型AIゲーム「淫らな尻とり」

1. プロジェクト概要

アプリ名: 淫らな尻とり

概要: プレイヤーがAIキャラクターとしりとりを行うWebアプリケーション。エッチな言葉を入力することでキャラクターの「欲情度」が上昇し、一定値（100%）に達すると勝利となる。

特徴:

Web Speech APIによる音声入力、またはテキスト入力による対話。

Google Cloud TTSを使用した高品質な音声読み上げ。

欲情度に応じた画像の視覚的変化（ぼかし・透過度の動的変更）。

Firebaseを使用した進行状況の自動セーブ・ロード。

2. システムアーキテクチャ

フロントエンド: React (SPA)

スタイリング: Tailwind CSS

アイコン: lucide-react

バックエンド (BFF): Vercel Serverless Functions (api/ ディレクトリ)

APIキーの隠蔽とCORS回避のため、フロントエンドからは直接外部APIを叩かず、Vercelの中継APIを経由する構成。

データベース・認証: Firebase (Firestore, Firebase Auth)

匿名認証またはカスタムトークンによる認証。

ユーザーごとの現在の欲情度や履歴を保存。

利用外部API:

テキスト生成・判定: Gemini 2.5 Flash API (Google AI Studio)

音声合成: Google Cloud Text-to-Speech API

3. 環境変数 (Vercel上に設定)

GEMINI_API_KEY: Google AI StudioのAPIキー

GCP_TTS_API_KEY: Google Cloud ConsoleのAPIキー（TTS用）

4. フォルダ・ファイル構成（主要部分）

/
├── api/
│   ├── chat.js  (Gemini APIとの通信を中継)

5. 主要な機能と仕様（App.jsx内）

パスコードロック (gameState === 'locked'):

アクセス時に合言葉（デフォルト: 0721）の入力が必要。

ホーム画面 (gameState === 'intro'):

遊び方（ヘルプ）、詳細設定への遷移。

セーブデータがある場合は「続きから」、ない場合は「開始する」。

開始時にマイクの使用許可を要求し、拒否された・非対応の場合は自動的に「テキスト入力モード (useTextInput = true)」へフォールバックする。

キャラクター選択 (gameState === 'character_select'):

3名のキャラクターから対戦相手を選択。設定画面からキャラクターの画像やプロンプトのカスタマイズが可能。

プレイ画面 (gameState === 'playing'):

音声モード: マイクアイコンをタップして発話。Web Speech API (SpeechRecognition) がテキスト化。

テキストモード: Inputフィールドからテキストを入力して送信。

入力テキストを /api/chat へPOST。AIがルール（既出単語NG、語尾「ん」NG）と欲情度変化を判定。

判定結果のセリフを /api/tts へPOSTし、Base64で受け取った音声を再生。

欲情度 (arousal) に応じて、背景画像のCSS filter (blur, brightness) と opacity を変化させる。

6. API中継コードの仕様

api/chat.js

役割: フロントエンドからシステムプロンプトとユーザー入力を受け取り、Gemini API (gemini-2.5-flash-preview-09-2025) にリクエストを送信。結果をJSONで返す。

認証: process.env.GEMINI_API_KEY を使用。

api/tts.js

役割: フロントエンドからテキストと音声設定を受け取り、Google Cloud TTS API (text:synthesize) にリクエストを送信。生成された音声データ(Base64)を含むJSONを返す。

認証: process.env.GCP_TTS_API_KEY を使用。

7. AIへの指示（開発・修正時の注意点）

このプロジェクトは 「フロントエンド(App.jsx) からは /api/chat と /api/tts をフェッチする」 という構成で完成しています。フロントエンドから直接外部APIを叩くように書き換えないでください。

コードの修正を提案する際は、原則として変更部分のみの差分ではなく、動作可能な完全なファイルのコードを出力してください。
