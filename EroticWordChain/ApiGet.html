APIキーの設定について

このWebゲームをフルに楽しむ（キャラクターとのAI会話や、フルボイスの音声読み上げ機能）ためには、Googleが提供する「APIキー」というパスワードのようなものが必要です。
このページでは、初心者の方でも迷わず取得できるように、手順を画像付きで解説します。

必要なもの（事前準備）

・Googleアカウント（普段お使いのGmailアドレスでOKです）
・クレジットカード（※Cloud Text-to-Speech取得時のみ、本人確認用として登録が必要です。無料枠内であれば課金はされません）

Gemini APIキーの取得手順

キャラクターとAI会話をするためのキーです。Google Cloud Consoleを使用して「Gemini API」を有効にし、キーを取得します。

1

Google Cloud Consoleにログイン

以下のリンクから「Google Cloud Console」にアクセスし、Googleアカウントでログインします。
Google Cloud Consoleを開く (別タブ)

初めて利用する場合は、利用規約の同意画面が表示されます。居住国（日本）を選択し、利用規約にチェックを入れて「同意して続行」を押します。

[画像: Cloud Console 初回同意画面] チェックボックスと「同意して続行」ボタンが強調されたスクリーンショット
2

プロジェクトを作成する

APIを利用するための「プロジェクト（箱のようなもの）」を作ります。
画面上部にある「プロジェクトの選択」をクリックし、右上の「新しいプロジェクト」を押します。
※既に他のAPI用にプロジェクトを作成している場合は、それを選択してStep3へ進んでください。

[画像: プロジェクトの作成画面] 画面上部のバーと、プロジェクト名入力欄が強調されたスクリーンショット
3

Gemini APIを有効にする

画面上部の検索バー（🔍 検索欄）に「Gemini API」と入力して検索します。
検索結果から「Gemini API」を選び、青い有効にするボタンを押します。

[画像: APIライブラリ画面] 「Gemini API」の「有効にする」ボタンが強調されたスクリーンショット
4

APIキーを作成してコピー

APIが有効になったら、左側のメニュー（≡アイコン）から
「API とサービス」＞「認証情報」をクリックします。

画面上部の「＋ 認証情報を作成」をクリックし、メニューから「APIキー」を選択します。

作成された長い英数字の文字列（APIキー）が表示されるので、コピーアイコンを押してコピーし、ゲームの設定画面に貼り付けてください。

[画像: APIキー生成完了画面] 「認証情報を作成」ドロップダウンと生成されたAPIキーが強調されたスクリーンショット

Cloud Text-to-Speech APIキーの取得手順

キャラクターのテキストを自然な音声で読み上げるためのキーです。少し手順が多いですが、順番通りに進めば大丈夫です。

1

Google Cloud Consoleにログイン

以下のリンクから「Google Cloud Console」にアクセスし、Googleアカウントでログインします。
Google Cloud Consoleを開く (別タブ)

初めて利用する場合は、利用規約の同意画面が表示されます。居住国（日本）を選択し、利用規約にチェックを入れて「同意して続行」を押します。

[画像: Cloud Console 初回同意画面] チェックボックスと「同意して続行」ボタンが強調されたスクリーンショット
2

プロジェクトを作成する

APIを利用するための「プロジェクト（箱のようなもの）」を作ります。
画面上部にある「プロジェクトの選択」をクリックし、右上の「新しいプロジェクト」を押します。

プロジェクト名に好きな名前（例：WebGameTTS）を入力し、「作成」を押します。

[画像: プロジェクトの作成画面] 画面上部のバーと、プロジェクト名入力欄が強調されたスクリーンショット
3

Text-to-Speech APIを有効にする

画面上部の検索バー（🔍 検索欄）に「Text-to-Speech API」と入力して検索します。
検索結果から「Cloud Text-to-Speech API」を選び、青い有効にするボタンを押します。

【重要】お支払い情報の登録について

APIを有効にする際、「課金を有効にしてください」と表示される場合があります。
Google Cloudの機能を使うにはクレジットカードの登録が必要です（本人確認のため）。
月間400万文字までは無料で利用できるため、個人でゲームを遊ぶ程度で課金されることはほぼありません。画面の指示に従い、請求先アカウントを設定してください。

[画像: APIライブラリ画面] 「Cloud Text-to-Speech API」の「有効にする」ボタンが強調されたスクリーンショット
4

APIキーを作成する

APIが有効になったら、左側のメニュー（≡アイコン）から
「API とサービス」＞「認証情報」をクリックします。

画面上部の「＋ 認証情報を作成」をクリックし、メニューから「APIキー」を選択します。

作成されたAPIキーが表示されるので、コピーアイコンを押してコピーし、ゲームの設定画面に貼り付けます。

[画像: 認証情報の作成画面] 「認証情報を作成」ドロップダウンと「APIキー」が強調されたスクリーンショット

よくある質問・トラブル

Q. APIキーを入力したのにエラーが出ます

A. コピーしたキーの前後に「余計な空白（スペース）」が入っていないか確認してください。
また、Cloud Text-to-Speechの場合、APIを「有効にする」手順を忘れていると機能しません。プロジェクトのライブラリからAPIが有効化されているか再確認してください。

Q. 勝手にお金がかかるのが不安です

A. Gemini API、Text-to-Speech APIともに広大な「無料枠」が設定されています。
個人のゲームプレイ程度で無料枠を超えることはありませんが、不安な場合はGoogle Cloudの「お支払い」メニューから「予算とアラート」を設定しておくことができます。また、遊ばなくなったらAPIキーを削除すれば安心です。

Q. APIキーを他の人に教えてもいいですか？

A. 絶対にダメです！
APIキーはあなた専用のパスワードと同じです。他人に教えたり、SNSにスクリーンショットを貼ったりすると、第三者に勝手に使われてしまう（課金される）恐れがあります。厳重に管理してください。
